Acerca del sistema

Sistema de Evaluación de Registros de Planeaciones Didácticas y Asignación de Materias 2026B. Consolida la información de 35 planteles del Colegio de Bachilleres del Estado de Durango para medir el cumplimiento institucional en el registro de planeaciones docentes en ces.cobaed.mx, contrastado contra la asignación de materias del RIA.

Los datos mostrados provienen del archivo RESULTADO_DEL_P45_VS_RIA.xlsx (929 registros). El botón "Sincronizar" intenta actualizar los datos desde la hoja de cálculo de Google Sheets vinculada al proyecto, cuando ésta se encuentra publicada en la web como CSV. Consulte el archivo README del repositorio para la guía de sincronización, despliegue como Web App de Google Apps Script y administración del sistema.
